## 快速开始

### 依赖检查
启动该项目前需要确保系统内有以下软件或资源：

1. k3s 1.27.7+k3s2
2. docker 24.0.6
3. image: uos:v0.1.0,nginx:alpine
4. java 17
5. k3s config


### 快速部署

通过docker快速部署

### 构建


1. 克隆源码

```bash
git clne https://github.com/jking412/backend-15.git
# 将k3s config文件放入backend-15目录下
mv path/config backend-15/config
```

2. 编译
```bash
cd backend-15
mvn install
```
